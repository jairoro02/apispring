# HEROES
## GET /HEROES/
Nos devuelve la lista de heroes
## GET /HEROES/1/
Nos devuelve el heroe con la id 1
## POST /HEROES/CREATE/
Crea un heroe con los datos introducidos
## PUT /HEROES/1/
Edita el heroe con id 1 con los datos introducidos
## DELETE /HEROES/1/
Borra el heroe con id 1
